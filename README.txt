# Projeto – Fase 1: Sistema Web do PetShop Feliz

Autor: Paulo Henrique Ferreira de Oliveira  
Disciplina: Fundamentos de Sistemas Web – Fase 1  

Este repositório contém a implementação da primeira fase do projeto de um sistema web simples para um petshop, utilizando apenas HTML, sem CSS ou JavaScript, conforme solicitado pela disciplina.

## Objetivo do Sistema
Desenvolver a estrutura inicial de um site para um petshop, apresentando produtos, serviços e navegação entre páginas.

## Estrutura do Sistema
O sistema possui três páginas HTML interligadas:

### 1. Home — index.html
- Apresentação do petshop.
- Introdução ao projeto.
- Navegação para Produtos e Serviços.

### 2. Produtos — produtos.html
Exibe produtos organizados em 3 categorias:
- Rações
- Brinquedos
- Acessórios

Cada item contém imagem, nome, descrição e preço.

### 3. Serviços — servicos.html
Serviços listados com:
- Nome
- Descrição
- Valor
- Indicação de tele-busca quando disponível

## Navegação
Todas as páginas possuem links para:
- Home
- Produtos
- Serviços
